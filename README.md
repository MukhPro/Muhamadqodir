# PDF Generator Bot

A Telegram bot that converts text into PDF files instantly and efficiently.

---

## Features

- Convert text into PDF format  
- Fast and lightweight performance  
- Simple Telegram bot interface  
- Works directly inside chat  
- Instant PDF file generation  

---

## How It Works

1. Start the bot in Telegram  
2. Send your text message  
3. The bot processes the content  
4. You receive a generated PDF file  

---

## Built With

- Python  
- Telegram Bot API  
- PDF generation library  

---

## Usage

Start the bot and send any text. The bot will automatically convert it into a downloadable PDF file.
